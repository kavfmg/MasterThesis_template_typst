修論用テンプレです

# usage
chapter/*.typにチャプターごとの記述をしてmain.typでincludeしてコンパイル。

見た目などはtheme内のファイルで主に記述してある。

## typst初心者向け
簡単に使い方をまとめています。

typstインストールして`typst compile main.typ output.pdf`を叩いてください。

(インストール方法は調べてください)
